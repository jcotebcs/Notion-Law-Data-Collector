Setup Instructions

1. Deploy This Application

  1. Deploy to Vercel, Railway, or Render
  2. Set the NOTION_API_KEY environment variable during deployment
  3. Get your Notion API key from Notion Integrations

2. Create a Database

  1. Create a new page in Notion
  2. Add a database with these properties:
    • Title (Title)
    • Case Number (Rich Text)
    • Court (Rich Text)
    • Judge (Rich Text)
    • Date (Date)
    • Status (Select)
    • Parties (Rich Text)
    • Type (Select)
    • Summary (Rich Text)
    • Outcome (Rich Text)
    • Tags (Multi-select)
    • Priority (Select)
  3. Share the database with your integration
  4. Copy the database ID from the URL

3. Configure This App

  1. Paste your database ID above
  2. Test the connection
  3. Start adding case data!
